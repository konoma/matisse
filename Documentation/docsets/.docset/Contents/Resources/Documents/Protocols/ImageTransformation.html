ImageTransformation

public protocol ImageTransformation: NSObjectProtocol

The ImageTransformation protocol describes objects that are capable of applying a transformation to a CGImage.

  • transformImage(_:)

    Apply the transformation to the given CGImage and return the transformed image.

  • Parameters:

    • image: The image to transform

    • Returns

    The transformed image.

    Parameters

    image

    Return Value

    The transformed image.

  • descriptor

    A string describing this transformation.

    The descriptor must describe this transformation such that equal transformations should have equal descriptions and different transformations must have different descriptors. The reason for this is that requests are considered equal based in part on wether their transformations are equal. This is checked using the descriptor.
