ImageRequestBuilder

public class ImageRequestBuilder: NSObject

ImageRequestBuilder provides a common interface for creating an ImageRequest.

The Swift and Objective-C DSL classes wrap this builder to provide their interface to API clients. This class encapsulates common logic, such as protecting against modifications after the request was built.
