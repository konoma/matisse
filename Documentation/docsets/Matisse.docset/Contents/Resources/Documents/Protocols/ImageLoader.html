ImageLoader

public protocol ImageLoader: class

An ImageLoader is responsible for downloading images.

For support in implementing an ImageLoader you can use the ImageLoaderBase as a parent class for your image loader.

  • loadImage(forRequest:completion:)

    Load the image for the given request.

    When the image was loaded, pass an URL of a temporary file that holds the downloaded image to the completion block. The caller takes ownership of this file and is responsible for deleting it when no longer in use.

    Note

    This method may be called from a background thread, but not concurrently.

  • Parameters:

    • request: The image request for which the image should be loaded.
    • completion: The completion block to call when the request has finished.
