DiskImageCache

public class DiskImageCache : ImageCache

A naive implementation of ImageCache that stores files on disk.

This class stores files in the caches directory of the application. It relies on the operating system to cleanup images. Suitable for a reasonable amount of images. If you download large numbers of images, you should probably look to implement this differently, or find a way to manage the directory size.

Images are serialized using NSKeyedArchiver to preserve file type and metadata.
